# Firebase Screenshot Repo (React + Firebase)

Starter project for an in-game screenshot repository using React + Firebase (Firestore + Storage + Functions).

## What’s included
- React frontend (components: Gallery, Upload, Winners, Navbar, ScreenshotCard, Admin)
- Firebase initialization (src/firebase.js)
- Cloud Function to pick “Best of the Week” (functions/index.js)
- Example Firestore rules and deployment steps

## Quick start
1. Copy `.env.example` to `.env` and fill your Firebase config.
2. `npm install`
3. `npm start`
4. Initialize Firebase in the project and deploy functions when ready.
